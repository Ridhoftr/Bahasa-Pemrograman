## Function Of Expectopn Handling

Exception handling adalah mekanisme dalam pemrograman untuk menangani error atau kondisi tak terduga yang muncul selama eksekusi program. Ini memungkinkan program untuk menangani error dengan cara yang terkendali tanpa harus berhenti tiba-tiba.

Di Python, exception handling dilakukan menggunakan blok try, except, else, dan finally.

- try: Blok ini digunakan untuk membungkus kode yang mungkin menyebabkan exception.

- except: Blok ini digunakan untuk menangani exception yang terjadi di blok try.

- else: Blok ini akan dieksekusi jika tidak ada exception yang terjadi di blok try.

- finally: Blok ini akan dieksekusi terlepas dari apakah ada exception atau tidak.
